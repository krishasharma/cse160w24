Implementation Overview

This project builds upon our previous work by adding advanced camera control and perspective projection in WebGL. Key features include interactive camera navigation, switching between orthographic and perspective views, and implementing view-dependent lighting effects.

  • Interactive camera control allowing navigation through the virtual world.
  • Ability to switch between orthographic and perspective projection.
  • Implementation of view-dependent lighting effects, including specular highlights.
  • Support for basic linear interpolation and advanced cubic curve interpolation in camera movements.

User Guide

To interact with the virtual world and explore different perspectives:

  1. Open the driver.html file in a modern web browser.
  2. Use the on-screen controls to navigate the camera and switch between different projection modes.
  3. Observe the changes in lighting effects as the camera moves or zooms.
