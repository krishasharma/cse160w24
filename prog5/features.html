Implementation Overview

This project involves creating an interactive 3D scene using Three.js. It features a detailed 3D heart model which users can interact with through various controls implemented in the web interface.

  • Development of a 3D heart shape using Three.js geometry.
  • Interactive controls allowing users to zoom in and out of the scene.
  • User toggle option for switching the scene's lighting on and off.
  • Responsive design adapting to different screen sizes and browser window adjustments.

User Guide

To interact with the 3D heart scene:

  1. Open the driver.html file in a modern web browser.
  2. Use the zoom slider to adjust your view of the heart model.
  3. Toggle the light using the provided checkbox to see the effect on the model.
  4. Use trackpad and pick outwards to resize scene if it renders too large.
  5. Resize the browser window to see the responsive adjustments of the scene.
