Implementation Overview

This project involves creating an interactive 2D drawing application using HTML5, JavaScript, and Canvas API. The application allows users to draw polylines on a canvas with specific features and functionalities.

  • Interactive polyline drawing with mouse clicks.
  • Canvas with displayed X-Y axes.
  • Coordinate logging in the console for each click.
  • End drawing feature with a right-click and final coordinate logging.

User Guide

To use the application:

  1. Open the driver.html file in a web browser.
  2. Left-click on the canvas to start drawing the polyline.
  3. Continue left-clicking to add more points to the polyline.
  4. Right-click to finalize the polyline and log the coordinates.
