Implementation Overview

This project involves the implementation of a 3D cube using WebGL. The cube can be rendered in both flat shading and wireframe modes, and users can interact with it through keyboard and slider controls.

  • Toggle between flat shading and wireframe rendering modes with the checkbox
  • Control cube rotation and scale using on-screen sliders.
  • Keyboard interaction for additional control.

User Guide

To interact with the 3D cube:

  1. Open the driver.html file in a web browser.
  2. Use the sliders to adjust the cube's rotation and scale.
  3. Press the 'W' key to toggle between wireframe and flat shading modes.
