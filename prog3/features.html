Implementation Overview

This project extends our exploration into WebGL, focusing on advanced lighting and shading techniques. The implementation covers different shading options (wireframe, flat shading, and smooth shading) and lighting effects (lambertian, specular, and ambient) along with the incorporation of directional and point light sources.

  • Implementation of Gouraud and Phong shading for smooth shading effects.
  • Integration of ambient and specular lighting alongside the existing lambertian lighting.
  • Ability to toggle directional and point light sources.
  • Movement of point light source represented by a sphere in the scene.

User Guide

To experience the different lighting and shading effects:

  1. Open the driver.html file in a modern web browser.
  2. Use on-screen controls to toggle between different shading modes and lighting effects.
  3. Adjust the position of the point light source using provided controls.
